# Generative AI Research Repository

This repository contains research materials for the project 

## Repository Structure

- literature_review  
  Contains academic papers and sources used for the literature review.

- quantitative_analysis  
  Contains survey data (CSV files), survey questions, and Python scripts used for quantitative analysis.

- qualitative_analysis
  Contains interview transcripts and qualitative research data.

- drafts_reports  
  Contains draft documents, proposals, and final reports.

- additional_materials  
  Contains images and other supporting media files.

- README
  Instructions about the folder.

## File Management

All files are organised based on data type to improve clarity and accessibility. Sensitive data, such as interview transcripts, should be handled carefully to ensure privacy and confidentiality.

## Collaboration Guidelines

Team members should:
- Use clear and consistent file names
- Avoid duplicate versions of files
- Write meaningful commit messages when making changes

## Notes

Some files may still need further organisation. Future updates will continue to improve the repository structure and naming conventions.
